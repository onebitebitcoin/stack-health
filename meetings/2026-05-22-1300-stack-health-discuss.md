## R1 의견

**기획**: 안건 1~3은 MVP 핵심과 직접 연관 낮음. 프로젝트명은 오늘 결정 필요. 댓글은 P1으로 런칭 후 1~2주 내 처리 권장.

**개발**: 안건1(30분), 안건2(1~2h), 안건3(3~4h), 안건4(4~6h). 댓글만 DB 마이그레이션 수반. 프로젝트명 변경은 런칭 후 권장.

**디자인**: Volt Dark 기본 찬성. 탭바 중앙 FAB(TikTok 패턴) 권장. 로고 SVG 모노크롬 제안. 댓글 바텀시트 70% 높이. Volt Light에서 `#B5FF2E` on 흰배경은 대비비 1.3:1 AA 미통과 — 액센트 전용 제한 필수.

**QA**: 댓글 P0 위험 — 모더레이션 5종(신고·관리자 숨김·레이트리밋·길이제한·로그인 필수) 미구현 시 비활성화 권고. 업로드 탭 제거 P1 — 대체 경로 확정 없이 제거 조건부 보류. Volt Light P1 — 토큰 누락 시 흰배경+흰텍스트 위험.

**악마의 변호인**: 6일 안에 4개 동시 불가. 업로드 탭 제거 — 콘텐츠 공급 동선 복잡화 근거 없음. Stack Health — 비트코인 색채 없음, 팬덤 연결 고리 단절. 댓글 모더레이션 비용 미계상.

**재무**: 테마 P1, 업로드 탭 전환율 리스크, Stack Health 보류(도메인·상표 미확인, SEO 자산 손실), 댓글 P2(운영 인건비 > 개발비).

**마케팅**: Stack Health 명칭 반대. 업로드 탭 제거 반대. 댓글 찬성(바이럴 루프).

**리서처**: 라임 색상 검증됨. FAB 패턴 TikTok 표준. STACK 피트니스 앱 이미 존재(`com.stackapp.fitness`). 댓글은 리텐션 핵심이나 콜드스타트 전략 필요.

**운영**: 댓글 모더레이션 SOP 없이 오픈 불가. 프로젝트명 변경 시 Railway·GitHub 연동 재확인 필요. 탭 제거와 대체 경로 단일 배포로 묶어야.

## R2 토론

**기획**: Stack Health 기각. Volt Dark GO. 탭 제거는 FAB 동시 배포 조건 충족 시 GO. 댓글 이번 배포 제외. "오늘 프로젝트명 결정" 촉구 — 리네임은 배포 체크리스트 5개 지점에 연쇄 영향.

**개발**: Volt Light `--accent-text: #65A30D`(lime-700, 4.6:1) 토큰 분기 방식 권장. 탭 제거+FAB: `BottomNav.tsx` 4탭→3탭, 중앙 FAB 별도 컴포넌트(2~3h). Stack Health 코드 변경 자체는 1h 미만이나 상표/도메인 확정 전 보류. 댓글 공수 R1 과소추정 확인 — 모더레이션 포함 7.5~9.5h. slowapi in-memory는 멀티 인스턴스에서 무력화(Redis 필요). 배포 묶음: 안건1→안건2→안건4(별도 배포)→안건3(확정 후). Railway 서비스명 연동 걱정은 과장 — `index.html` title/meta 15분 작업.

**디자인**: Volt Light 토큰 구조 확정: `color-accent-surface` (#B5FF2E), `color-accent-on-surface` (#1A1A1A), `color-accent-emphasis` (다크 배경 위 #B5FF2E). FAB 스펙 확정: 56dp 원형, 탭바 상단 경계 -28dp, lucide `Plus` 24dp, #B5FF2E 배경 + #1A1A1A 아이콘. Stack Health 보류(로고·OG·favicon 전체 재작업 6일 내 처리 시 P0 압박). 댓글 바텀시트: 40%/70%/full 스냅포인트, 키보드 회피, 롱프레스 신고·숨김 메뉴.

**QA**: Volt Dark GO. Volt Light 조건부 GO (AA 토큰 명세 확정 선행). 탭+FAB 조건부 GO (E2E 경로 수정 선행). 댓글 HOLD (모더레이션 P0 선결).

**악마의 변호인**: Stack Health 반대 근거 "비트코인 색채 없음"은 감정적 주장 아님 — 브랜드 연결 고리 단절은 초기 팬덤 확보에 실질 리스크. 탭 제거 "대체 경로" 명세가 아직 운영 수준으로 구체화되지 않음. 댓글 "런칭 후 처리"는 회피이지 해소가 아님 — 언제, 누가 SOP를 완성할지 책임자가 없음.

**재무**: Volt Dark GO (비용 0). 탭 제거 조건부 GO (대체 경로 보장 시). Stack Health 보류 유지 — 상표 조회 비용은 0이나 미조회 상태에서 확정하면 법적 비용 리스크. 댓글 런칭 후 P2 유지. 비트코인 리워드 재원 비율 미결 — 이것 없이는 손익분기 계산 불완전 (🔴 즉시 결정).

**마케팅**: Stack Health 반대 유지 (검색 희석, 비트코인 연결 부재). 대안 브랜드명 탐색 권고 (비트코인·운동·인센티브를 함께 담는 방향). 탭 제거 조건부 반대 (탭 가시성 데이터 vs FAB UX 데이터 상충). 댓글 P1으로 상향 권고 (바이럴 루프 핵심).

**리서처**: STACK 앱(`com.stackapp.fitness`) Google Play 존재 확인. App Store에도 Stacked 3종 확인. "Stack Health" 상표 USPTO·KIPRIS 직접 조회 필요. `stackhealth.com` 도메인 가용성 미확인. 댓글 리텐션 데이터 지지: Strava 소셜 기능 후 세션 35회/월, 소셜 피처 앱 리텐션 30% 향상(2024-2025 벤치마크). 탭 제거 리스크 데이터: Zeebox 탭 제거 후 참여도 급락 사례.

**운영**: 배포 묶음 분리안: **배포 A** (Volt Dark 전환 + 탭제거+FAB + 리네임) / **배포 B** (댓글, 선결 조건 충족 후 별도). 탭 제거와 FAB는 반드시 동일 PR에 묶어야 — 분리 배포 시 경로 단절 윈도우 발생. 댓글 최소 SOP 3종: (a) 신고 버튼 UI, (b) 운영자 검토 큐, (c) 72h 내 처리 기준. 씨앗 댓글 계정 관리 절차도 SOP에 포함.

### ⚠️ 악마의 변호인 미해소 경고

1. **Stack Health — 비트코인 팬덤 연결 고리 단절**: "Stack"은 피트니스 앱 네임스페이스에 이미 포화 상태이며, 비트코인·라이트닝 네트워크 커뮤니티와의 연결 고리가 없다. 대안 브랜드명 탐색 없이 채택/보류만 반복되고 있음. → **미해소**

2. **댓글 모더레이션 비용 — 책임자 없음**: "런칭 후 SOP 완성"을 여러 역할이 제안했으나, 누가 SOP를 작성하고 언제까지인지 오너가 지정되지 않음. → **미해소**
